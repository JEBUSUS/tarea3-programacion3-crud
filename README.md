# tarea3-programacion3-crud

Tarea 3 - Programación III - CRUD con Git Flow



Este repositorio contiene un CRUD simple de tareas hecho con HTML, CSS y JavaScript.

Se utiliza para la Tarea 3 de la asignatura Programación III, practicando el uso de

Git y Git Flow (ramas main, dev, qa y ramas feature/hotfix).



&nbsp;Funcionalidades actuales



\- Crear tareas con título, descripción y estado.

\- Listar las tareas en una tabla.

\- Editar tareas existentes.

\- Eliminar tareas.
